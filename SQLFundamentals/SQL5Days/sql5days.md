# 5-Day SQL Curriculum

## Day 1: Introduction to RDBMS & SQL Basics

- Introduction to databases & RDBMS
- Data models in databases
- Properties of RDBMS
- CODD’s Relational Database Rules
- Data integrity concepts
- Introduction to T-SQL language
- Working with data types (basics)
- Working with schema and tables
- DDL, DML, DCL statements
- Implementing data integrity

## Day 2: Core SQL & Data Manipulation

- Deep dive into T-SQL
- System functions
- Advanced T-SQL queries (SELECT, WHERE, GROUP BY, HAVING, ORDER BY)
- Advanced T-SQL statements (UPDATE, DELETE, INSERT)
- Other T-SQL statements (TRUNCATE, MERGE, etc.)
- Set operators (UNION, INTERSECT, EXCEPT)
- Hands-on exercises

## Day 3: Joins, Subqueries, and Data Retrieval

- What are joins?
- Types of joins (INNER, LEFT, RIGHT, FULL, CROSS)
- Subqueries (correlated and non-correlated)
- Practical scenarios using joins and subqueries
- Complex query building
- Query optimization basics

## Day 4: Database Objects, Indexes, Views, and Stored Procedures

- Introduction to indexes in SQL Server
- Creating and using indexes
- Introduction to views in SQL Server
- Creating and querying views
- Stored procedures: concepts and implementation
- Exception handling using TRY-CATCH
- Hands-on: writing and using stored procedures, views, and indexes

## Day 5: NoSQL & Cloud Databases (Azure SQL)

- Brief history of NoSQL databases
- NoSQL database features and types
- Differences between RDBMS and NoSQL
- When and why to use NoSQL
- NoSQL demo (basic CRUD operations)
- Introduction to Azure SQL Database
- Demo: Creating and managing Azure SQL Database
- Review, Q&A, and wrap-up

## End. Next Steps.

Practice. Practice. Practice. Also, learn basic set theory from high school math.
